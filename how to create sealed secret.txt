How to create sealed secrets:
1. Get the Current Cluster's Public Certificate
Bash
Copy code
kubeseal --fetch-cert \
  --controller-namespace=sealed-secrets \
  --controller-name=sealed-secrets-controller \
  > sealed-secrets-cert.pem
2. Create the Original Secret
Bash
Copy code
cat <<EOF > dynatrace-secret.yaml
apiVersion: v1
kind: Secret
metadata:
  name: ocp1-dev-pdc
  namespace: dynatrace
type: Opaque
stringData:
  apiToken: "<YOUR_DYNATRACE_API_TOKEN>"
  dataIngestToken: "<YOUR_DYNATRACE_DATA_INGEST_TOKEN>"
EOF
3. Seal the Secret with This Cluster's Key
Bash
Copy code
kubeseal --cert sealed-secrets-cert.pem \
  --format yaml \
  < dynatrace-secret.yaml \
  > dynatrace-sealedsecret.yaml
4. Delete the Old SealedSecret and Apply the New One
Bash
Copy code
oc delete sealedsecret ocp1-dev-pdc -n dynatrace - Skip
oc apply -f dynatrace-sealedsecret.yaml  ---------------------> will do from Argocd
